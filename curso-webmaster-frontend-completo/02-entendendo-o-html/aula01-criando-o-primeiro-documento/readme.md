# Criando o primeiro documento

- Dentro da nossa pasta, criar um arquivo chamado `index.html`

- Abrir o arquivo criado com o editor de texto de preferência

## Configurando o arquivo Index

- Com o arquivo index aberto no editor de texto, vamos acrescentar as seguintes TAGs

```html
html

<!-- Identifica que o conteúdo é do tipo HTML -->
<!DOCTYPE html>

<!-- head é a tag que contém metadados e links -->
<head>
    <!-- METADADOS -->

    <!-- Tag que corrige a acentuação dos textos do website -->
    <meta http-equiv="Content-Type" content="text/html;charset=UTF-8">

    <!-- LINKS -->
    
    <!-- Tag que informa a localização do arquivo CSS -->
    <link rel="stylesheet" href="style.css">

    <!-- Tag que define o título do site -->
    <title>Título do meu website</title> 
</head>

<!-- body é a tag que vai todo conteúdo visual do website -->
<body>
    
</body>
</html>
```
